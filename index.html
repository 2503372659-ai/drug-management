使用指南
按这个流程录入，提醒才会准确。
1先添加患者

填写姓名、电话、医保类型、常去医院和备注。患者资料越完整，后面搜索和回访越方便。

2再添加药品

进入患者卡片，点击“添加药品”。药品名、当前总量、每日用量、开始计算日期是提醒计算的关键。

3查看今日待办

首页会自动列出已超期、临近取药、待补药品的患者。你可以用“全部 / 已超期 / 临近 / 待补药品”快速筛选。

4用日历集中安排

日历会按预计取药日期展示每位患者的药品。点击日历中的事项，可以直接进入患者工作台继续处理。

5进入患者工作台

患者卡片中的“详情”会打开独立工作台，可统一查看患者资料、所有药品、风险状态和取药时间线。

6取药后要记录

点击“记录取药”，填写本次新取药数量和日期。系统会把剩余药量加上新药量，并从取药日重新计算下次日期。

7换药不要直接删

药品暂时不用时选“暂停”，以后不再用时选“已停用”。这样历史还能保留，也不会继续进入提醒。

8定期导出备份

数据保存在当前浏览器。换电脑、清浏览器或重装系统前，请在设置里导出 JSON 备份。

计算规则：预计下次取药日期 = 开始计算日期 + 当前总量 ÷ 每日用量。暂停和停用药品不会提醒。OCR 识别只能辅助录入，药品数量和每日用量请人工确认。
